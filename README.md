## 个微风控
#### 【微信风控规则】
- 新设备登录微信， 24小时（1天）内不能发、评论朋友圈，部分账号72小时内不能评论朋友圈。
- 朋友圈点赞和评论单天上限建议500次。
- 主动添加好友单天上限30个，超过后对方会收不到请求，推荐每天最多加15-20个，1小时内不超过5个；
- 拉人入群频率不要太高，新微信号不要建群和拉人，一个群1小时拉人建议不超过10个，单次最多拉40人（微信限制）；
- 一天可以被申请添加200-300人左右，一些被风控的号会少于200；
- 群内加好友每小时建议不超过5个，每天不超过30个；
- 新微信号一定要养一段时间号！！
- 群发推送
  - 推荐使用智能体生成推送话术，这样可以避免连续发同质化的推送内容；
  - 批量推送，单次最好不超过40人，如果人数很多，可以设置多个推送任务（机器人产品会提供策略规则）；
  - 两次推送之间，推荐间隔15-30分钟；

#### 【关于养号】
新号或者微信号换了新设备需要养号：新号至少养7天以上，进行正常聊天、发朋友圈、完成实名认证、一些小额支付行为等，避免一上线就做大批量操作

#### 【内容合规】
避免敏感词、骗人的营销、强推性质的群发，控制单个联系人一段时间内接收消息的次数。被举报是直接触发人工或算法审查的重要原因
